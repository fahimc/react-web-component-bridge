# React To Web Component Conversion Examples

This directory contains ten side-by-side examples of React-shaped component source and the no-React Custom Element JavaScript generated by `@codedia/react-to-web-component-compiler`.

The generated output is split deliberately:

- [converted/scaffold.js](./converted/scaffold.js) contains the shared compiler runtime scaffold.
- Each `converted/*.js` component file imports from that scaffold and contains only the converted component plus its custom-element registration.

Regenerate the converted files after compiler changes:

```bash
pnpm build
node examples/conversions/generate.mjs
```

## Example Pairs

| React source                                     | Converted Custom Element output                    | Demonstrates                                     |
| ------------------------------------------------ | -------------------------------------------------- | ------------------------------------------------ |
| [metric-card.tsx](./react/metric-card.tsx)       | [metric-card.js](./converted/metric-card.js)       | `useMemo`, props, CustomEvent callback           |
| [toggle-switch.tsx](./react/toggle-switch.tsx)   | [toggle-switch.js](./converted/toggle-switch.js)   | `useState`, boolean attributes, click events     |
| [customer-list.tsx](./react/customer-list.tsx)   | [customer-list.js](./converted/customer-list.js)   | arrays as properties, filtering, mapped children |
| [counter-panel.tsx](./react/counter-panel.tsx)   | [counter-panel.js](./converted/counter-panel.js)   | `useReducer`, number attributes                  |
| [alert-banner.tsx](./react/alert-banner.tsx)     | [alert-banner.js](./converted/alert-banner.js)     | default props, tone variants, dismiss event      |
| [task-board.tsx](./react/task-board.tsx)         | [task-board.js](./converted/task-board.js)         | grouped lists, object events                     |
| [profile-card.tsx](./react/profile-card.tsx)     | [profile-card.js](./converted/profile-card.js)     | `useId`, accessible labels                       |
| [pricing-card.tsx](./react/pricing-card.tsx)     | [pricing-card.js](./converted/pricing-card.js)     | `memo`, reflected props                          |
| [filter-toolbar.tsx](./react/filter-toolbar.tsx) | [filter-toolbar.js](./converted/filter-toolbar.js) | `useDeferredValue`, selected state               |
| [async-status.tsx](./react/async-status.tsx)     | [async-status.js](./converted/async-status.js)     | `lazy`, `Suspense`, async rerender               |

Each converted file is intentionally committed so npm and GitHub readers can inspect the exact JavaScript shape without running the build.
